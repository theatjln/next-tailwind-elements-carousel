Tailwind Elements Reference: https://tailwind-elements.com/quick-start/

React Helmet Reference: https://github.com/nfl/react-helmet
(React-Helmet - This reusable React component will manage all of your changes to the document head.
Helmet takes plain HTML tags and outputs plain HTML tags. It's dead simple, and React beginner friendly.)

1. npm i react-helmet and import { Helmet } from "react-helmet"; to the component where your carousel will render

2. import { useEffect } from "react"; to the component where your carousel will render, create a useEffect function inside the component, and add these codes inside the useEffect function:
    var aScript = document.createElement("script");
    aScript.type = "text/javascript";
    aScript.src = "https://cdn.jsdelivr.net/npm/tw-elements/dist/js/index.min.js";

    document.head.appendChild(aScript);
    aScript.onload = () => {};

3. create a React Fragment to be returned by the component, then paste this code inside that React Fragment:
